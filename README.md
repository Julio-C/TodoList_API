# TodoList_Api
É uma API RESTFUL simples para gerenciar tarefas.

Version 1.0.0

Deploy da aplicação:

Para realizar o deploy dessa api, será necessário ter instalado uma IDE JAVA e o POSTGRES.
Para a construção da API utilizei o Spring Tools 4.

Baixe o TodoList_Api
importe como um projeto maven
o projeto esta configurado para executar na porta 8080.

Postgres:

Criar o DataBase tarefas-apirest, que é onde a api criara as tabelas automaticamente após a execução.
Configuração:
		Localhost : http://127.0.0.1
		PORT : 5432
		username=postgres
		password=postgres

Foram realizados testes com o Postman e o Curl, a seguir os endpoints para realização de testes.
